Calculadora de Payback (V1)

Meu primeiro projeto na Teison Brasil, desenvolvido ainda como estagiário. Trata-se de uma calculadora que, a partir de diversos cálculos elaborados pelo engenheiro, estima o tempo necessário para o retorno do investimento. A ferramenta é amplamente utilizada pelos vendedores para embasar negociações com clientes, tornando o processo mais estratégico e assertivo.
