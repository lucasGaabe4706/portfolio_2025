Explicação Flexbox

Propriedades do Container (Pai)
Essas propriedades controlam o comportamento geral do layout dos itens dentro do container:

display: flex;

Ativa o Flexbox no container (obrigatório).
flex-direction

Define a direção dos itens (linha ou coluna).
row → Horizontal (padrão).
column → Vertical.
row-reverse → Horizontal invertido.
column-reverse → Vertical invertido.
justify-content

Alinha os itens no eixo principal (horizontal por padrão).
flex-start → Alinha os itens no início.
flex-end → Alinha os itens no final.
center → Centraliza os itens.
space-between → Deixa espaço igual entre os itens.
space-around → Deixa espaço igual em torno de cada item.
align-items

Alinha os itens no eixo perpendicular (vertical por padrão).
stretch → Expande os itens para preencher o container (padrão).
center → Centraliza os itens verticalmente.
flex-start → Alinha no topo (ou à esquerda em flex-direction: column).
flex-end → Alinha na parte inferior (ou à direita em flex-direction: column).
flex-wrap

Define se os itens devem quebrar linha caso não caibam no container.
nowrap → Não permite quebra (padrão).
wrap → Permite quebra.
wrap-reverse → Permite quebra na direção oposta.
align-content

Alinha várias linhas (quando flex-wrap está ativado).
flex-start, flex-end, center, space-between, space-around.
Propriedades dos Itens (Filhos)
Essas propriedades controlam o comportamento individual de cada item dentro do container:

flex-grow

Define quanto o item pode crescer para preencher o espaço disponível.
0 → Não cresce (padrão).
1 → Cresce proporcionalmente ao espaço disponível.
flex-shrink

Define quanto o item pode encolher se o espaço for insuficiente.
1 → Encolhe proporcionalmente (padrão).
0 → Não encolhe.
flex-basis

Define o tamanho inicial do item antes do ajuste com flex-grow ou flex-shrink.
Pode ser especificado em pixels, %, etc.
align-self

Alinha um item individualmente, ignorando align-items do container.
auto → Herda do container (padrão).
flex-start, flex-end, center, baseline, stretch.
order

Define a ordem de exibição dos itens (valor padrão é 0).
Itens com menor valor aparecem primeiro.
